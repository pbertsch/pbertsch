# Patrick Bertsch

**German software engineer and co-founder — building developer tooling and on-device AI from Guadalajara, México.**

Co-Founder of [Alpha Wave Systems](https://alphawave.systems), a Mexican software studio. I work full-stack — Flutter for mobile and web, Go and Node/TypeScript on the backend — and I'm happiest shipping small, sharp tools. Hungry for knowledge, always up for a new crazy project.

### What I'm building — public repos

- **[flutter-probe](https://github.com/AlphaWaveSystems/flutter-probe)** — Flutter E2E testing framework: write tests in plain English and run them against the live widget tree with sub-50ms round-trips. An open alternative to `integration_test` and Patrol; ships with its own MCP server and a [Homebrew tap](https://github.com/AlphaWaveSystems/homebrew-tap). *(Go · Dart)*
- **[tqai](https://github.com/AlphaWaveSystems/tqai)** — TurboQuant KV-cache compression for local LLM inference: ~80% memory savings with near-zero quality loss on 8B+ models. *(Python · MLX, Apple Silicon)*
- **AWSYS.CO SDKs** — official open-source client libraries for the [AWSYS.CO](https://awsys.co) API, with full type safety and automatic retry: [TypeScript](https://github.com/AlphaWaveSystems/awsysco-ts-sdk) (`@awsysco/sdk`) · [Python](https://github.com/AlphaWaveSystems/awsysco-python-sdk) (`awsysco`) · [Go](https://github.com/AlphaWaveSystems/awsysco-go-sdk).
- **[e2e-test-workspace](https://github.com/AlphaWaveSystems/e2e-test-workspace)** — a multi-stack reference workspace (Flutter + React + Go) for end-to-end testing. *(Dart)*
